五行与十神理论基础

五行学说是中国古代的一种基本哲学观，认为世界由木、火、土、金、水五种基本物质构成。五行之间存在相生相克的关系：

  • 相生关系：木生火、火生土、土生金、金生水、水生木
  • 相克关系：木克土、土克水、水克火、火克金、金克木

十天干（甲乙丙丁戊己庚辛壬癸）各自属于五行中的一种，并且每种五行有阴阳之分：

五行 阳干 阴干 基本特性
木 甲 乙 生长、向上、柔韧
火 丙 丁 炽热、光明、上升
土 戊 己 厚重、包容、承载
金 庚 辛 刚硬、锐利、收敛
水 壬 癸 寒冷、向下、流动

十神形成原理：十神是日主（八字中的日干）与其他天干的关系。基于五行相生相克和阴阳关系，形成十种不同的关系，这就是十神。

十神形成的计算方法

1. 同五行关系：当天干与日主五行相同时

  • 若阴阳相同 → 比肩
  • 若阴阳不同 → 劫财

2. 生我关系：当天干五行生日主五行时

  • 若天干为阳 → 偏印
  • 若天干为阴 → 正印

3. 克我关系：当天干五行克日主五行时

  • 若天干为阳 → 七杀
  • 若天干为阴 → 正官

4. 我生关系：当日主五行生天干五行时

  • 若天干为阳 → 食神
  • 若天干为阴 → 伤官

5. 我克关系：当日主五行克天干五行时

  • 若天干为阳 → 偏财
  • 若天干为阴 → 正财
